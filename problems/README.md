# Problems

Each directory contains one long-running mathematical problem or tightly related research programme.

A problem directory begins with:

- `README.md` — scope, current phase, canonical landscapes, and navigation;
- `STATUS.md` — current position, blockers, and next decision;
- `CLAIMS.md` — authoritative claim ledger.

It may also contain:

- `foundations/` — definitions, models, and established results shared across investigations;
- `references/` — canonical literature landscapes, testbed maps, and source indexes;
- `reductions/` — reusable reduction infrastructure;
- `investigations/` — concrete problem environments;
- route-local proofs, counterexamples, audits, experiments, notes, and archives.
