# Problems

Each directory contains one long-running mathematical problem or tightly related research program.

A problem directory begins with:

- `README.md` — scope, definitions, and navigation;
- `STATUS.md` — current position, blockers, and next action;
- `CLAIMS.md` — authoritative claim ledger.

Additional notes, proofs, counterexamples, experiments, references, and archived routes should be added only when needed. Avoid duplicating repository-wide standards inside each problem.
