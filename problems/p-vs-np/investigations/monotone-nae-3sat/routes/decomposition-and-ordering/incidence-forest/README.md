# R2.2 — Incidence-Forest Decomposition

**Subroute:** `R2.2`  
**Status:** `PROVED / RESTRICTED`  
**Primary claim:** `NAE-008 — PROVED / CHECKED`

## Result

Every finite three-uniform hypergraph whose incidence graph is a forest is two-colourable, and a valid colouring is constructible in linear incidence-graph time.

## Scope

This is a constructive tractable-class theorem. It does not extend automatically to bounded-cycle, bounded-feedback or arbitrary sparse incidence graphs.

## Role in the programme

- positive decomposition control;
- base case for exact gluing systems;
- calibration target for symbolic representations and propagation methods;
- starting point for determining how cycles reintroduce global compatibility.

## Next extension question

Identify the largest precisely defined class beyond incidence forests for which exact gluing or residual state remains polynomially bounded.

Any extension must account for the number and interaction of cycle-induced compatibility choices.
