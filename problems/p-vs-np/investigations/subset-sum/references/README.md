## Source policy

Prefer primary sources. Each entry should identify the exact theorem, construction, algorithm, or lower bound used, with page, theorem, or stable section information when available.

Do not copy external results into the repository claim ledger as `PROVED`; use `ESTABLISHED` with a primary citation when an external result changes the accepted mathematical state.

Literature maps and route-selection matrices do not by themselves change claim status.
