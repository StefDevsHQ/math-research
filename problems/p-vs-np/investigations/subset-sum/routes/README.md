# Research Routes

Each route is a distinct proposed mechanism, algorithmic strategy, or obstruction program.

Every active route should contain:

- `README.md` — thesis, scope, and dependencies;
- `STATUS.md` — current operational state and next test;
- `CLAIMS.md` — route-local claim ledger;
- atomic directories for proofs, counterexamples, audits, experiments, notes, and archive.

Do not merge routes merely because they use related vocabulary. Preserve failed routes when they remain informative.
