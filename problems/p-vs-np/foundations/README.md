# Foundations

Shared definitions, complexity conventions, and established results for the P versus NP program live here.

Use this directory for material that applies across multiple investigations:

- formal problem and complexity-class definitions;
- encoding and input-length conventions;
- deterministic and nondeterministic computation models;
- externally established results with primary citations.

Do not place route-specific arguments here. New repository claims belong in the nearest investigation or route ledger.
