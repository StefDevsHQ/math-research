# P versus NP

This directory contains a disciplined investigation of the P versus NP problem and selected NP-complete problems used as concrete research environments.

The repository does not claim a resolution. Its purpose is to preserve exact statements, proof attempts, failed routes, counterexamples, computational evidence, and the current research frontier without conflating them.

## Working scope

The current program studies whether exact algorithms for NP-complete problems admit structural compression that is polynomial in the binary input length.
