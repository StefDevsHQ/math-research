# Reductions

Canonical reductions shared by more than one investigation live here.

Each reduction directory should record:

- source and exact theorem used;
- input and output encodings;
- correctness argument or primary citation;
- size and time bounds;
- assumptions such as base choice or no-carry conditions;
- audits and known implementation hazards.

A reduction is infrastructure. Conclusions drawn from applying it belong in the relevant investigation or route.
