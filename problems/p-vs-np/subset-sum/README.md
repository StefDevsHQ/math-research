# Subset Sum

Subset Sum is the first concrete research environment in the P versus NP program.

Given positive integers

\[
A=\{a_1,\ldots,a_n\}
\]

and a target \(T\), determine whether some subset of \(A\) sums exactly to \(T\).

## Research focus

The current investigation asks whether exact reachable-sum states can be represented and updated using size polynomial in the binary input length.

The first completed route concerns structural compression through dense regions, separated choices, arithmetic progressions, residue completion, and recursive lattice structure. Its canonical record will be imported after the repository foundation is established.
