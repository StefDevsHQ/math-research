# Research Tools

Reproducible scripts, notebooks, and small utilities used to test or verify research claims belong here.

Each tool must identify:

- the claim or question it tests;
- its inputs and output format;
- dependencies and versions;
- the exact command or entry point;
- limitations of the computation.

Executable material in this directory is licensed under the [MIT License](LICENSE). Generated results belong with the relevant investigation under its `experiments/` directory, not in this directory.
